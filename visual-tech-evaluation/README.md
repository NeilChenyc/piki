# Piki 官网视觉技术方案评估

> 评估日期：2026-07-04  
> 评估目标：为 Piki 产品官网找到 AI 可快速生成的高质量视觉/动画方案

---

## 当前状态

Piki 官网（`website/`）使用 **Astro 5.8** 构建，所有视觉均为**手写内联 SVG + 纯 CSS @keyframes 动画**，通过 IntersectionObserver 驱动滚动入场。

**优点**：零依赖、性能极佳、风格统一  
**痛点**：SVG 全靠手写、视觉风格单一（绿色单色系白卡风）、无复杂动画、缺少产品感

---

## 评估维度

| 维度 | 说明 |
|------|------|
| 🎨 视觉效果 | 1-5 分，产出能达到的视觉上限 |
| 🤖 AI 生成难度 | 越容易让 AI 直接生成代码/文件越好 |

| 🔗 Astro 兼容 | 与现有 Astro 技术栈的整合难度 |

# 二、矢量动画平台

## 5. LottieFiles AI ⭐推荐

**官网**：https://lottiefiles.com/ai

**能力**：
- **AI Vector Generator**：文字描述 → 高质量 SVG 矢量图
- **AI Animation**：SVG → Lottie 动画
- 海量免费动画库可直接下载使用
- Lottie 运行时仅 ~30KB，JSON 格式

| 维度 | 评分 |
|------|------|
| 🎨 视觉效果 | ⭐⭐⭐⭐⭐ (5/5) — 专业级矢量动画 |
| 🤖 AI 生成难度 | ⭐⭐⭐⭐ (4/5) — 文字生成 SVG，再手动加动画 |
| ⚡ 性能影响 | ⭐⭐⭐⭐⭐ (5/5) — Lottie 轻量，比 GIF 小 5-10x |
| 🔗 Astro 兼容 | ⭐⭐⭐⭐ (4/5) — 引入 lottie-web/lottie-light 即可 |
| 💰 费用 | 免费额度充足，Pro $12/月 |
| 🎯 官网适用 | Hero 动画、功能图标动效、加载动画、微交互 |

**工作流**：
1. LottieFiles AI 文字描述 → 生成 SVG 矢量
2. Lottie Creator 加动画 → 导出 Lottie JSON
3. 网站引入 `lottie-web` 或 `<dotlottie-player>`
4. AI 也可以直接搜索 LottieFiles 免费库使用现成动画

# 六、策略性方案

## 17. 增强版 AI 直出内联 SVG + CSS Animation（当前方式升级）⭐推荐

**思路**：保持当前 Astro 零依赖风格，但让 AI 生成更复杂的内联 SVG + CSS 动画。

**升级方向**：
- AI 生成更丰富的渐变、阴影、滤镜效果
- CSS `@property` 实现自定义属性动画
- CSS `scroll()` / `view()` 原生滚动驱动动画（Chrome 115+）
- SVG `<animate>` / `<animateTransform>` 元素动画
- SVG `filter` 效果（毛玻璃、发光、噪点纹理）

| 维度 | 评分 |
|------|------|
| 🎨 视觉效果 | ⭐⭐⭐⭐ (4/5) — 有上限但够用 |
| 🤖 AI 生成难度 | ⭐⭐⭐⭐⭐ (5/5) — WorkBuddy 原生擅长 |
| ⚡ 性能影响 | ⭐⭐⭐⭐⭐ (5/5) — 零 JS 开销 |
| 🔗 Astro 兼容 | ⭐⭐⭐⭐⭐ (5/5) — 完美兼容 |
| 💰 费用 | 完全免费 |
| 🎯 官网适用 | Hero 图解、Feature 插图、全站视觉 |
