# 大模型维基

## 来源

- 路径：`raw/sources/llm-wiki.md`
- 作者：本地对话中归因于安德烈·卡帕西的大模型维基想法
- 日期：本地来源未注明
- 格式：Markdown 想法文档

## 摘要

大模型维基是一种个人知识库模式：大模型从不可变的原始来源中持续维护一组相互链接的 Markdown 页面。它不同于普通检索增强生成，不是在每次查询时重新从零散文档中召回和综合，而是先把每个来源编译进持久页面：来源摘要、概念、实体、领域、矛盾和综合。

这个来源把知识库分成三层：原始来源是真相来源，维基是可维护的编译层，`AGENTS.md` 是告诉后续大模型如何操作的结构约定。

## 关键想法

- 维基会复利，因为结构和综合跨会话保留下来。
- 大模型负责繁琐维护：摘要、归档、链接、更新和一致性检查。
- 人负责来源策展、探索方向、判断和问题选择。
- [[index]] 是内容目录，[[log]] 是时间线记录。
- 好的查询答案可以回存为综合页面。
- 定期检查可以发现矛盾、过期说法、孤立页面和缺失概念。

## 重要细节

这个架构包含三层：

- 原始来源：不可变的文档、文章、图片、笔记和其他输入。
- 维基：由大模型生成和维护的 Markdown 页面。
- 结构约定：定义页面格式、链接方式和工作流的规则文件。

这个来源建议先轻量启动。中等规模的维基可以先依赖索引、本地搜索和 Markdown 链接，不必过早引入向量检索基础设施。
