# 黑曜石

## 这是什么

黑曜石是一款本地 Markdown 知识库应用。种子来源把它作为浏览和使用维基的良好环境。

## 相关性

大模型维基模式把黑曜石看成维基的阅读和浏览环境：大模型编辑 Markdown 文件，用户浏览页面、追踪链接并查看图谱。
