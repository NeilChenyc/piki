# 微信产品哲学

微信产品哲学是 **张小龙** 在打造微信及其他产品（Foxmail、QQ 邮箱）过程中形成的一套系统化产品方法论。其核心是 **从人性出发**，将产品视为有灵魂的作品而非工具。

---

## 核心理念

### 1. 人性驱动设计

- **接受并利用人的弱点**：懒惰、贪嗔痴（贪、嗔、痴——佛教三毒）、缺乏耐心、不爱学习
- **用户不是理性"经济人"**, 而是被环境、情感、群体心理驱动的"人"
- **通过理解自己来理解用户**（"以己推人"）——不做"已读"是因为自己不想被知道已读
- **人是环境的反应器** — 产品要营造"环境"来牵引用户行为

### 2. 群体效应优先

- 互联网产品面对的是 **群体而非个体**
- 产品应 **建立规则和环境**，让群体在其中自我演化
- **不可预知性和不可测性是产品的生命力所在**
- 典型案例：
  - **"摇一摇"** — 同一时刻做同一动作的群体效应，满足随机性好奇心
  - **"附近的人"** — 满足对周围环境的好奇心（不是为交友）
  - **"漂流瓶"** — 满足倾诉和发泄欲望
- 群体行为无法通过个体调研推导（引用勒庞《乌合之众》和凯文·凯利《失控》）

### 3. 极简主义

- **"极简方能不被超越"** — "摇一摇"简化到竞争对手无法超越
- **分类和抽象是设计最核心的方法** — 100 个需求抽象为 10 个再派生
- **"宁愿损失功能也不损失体验"** — 不做"编辑"按钮防误触返回
- **做"类型"而非做"实例"** — 订阅平台抽象为一种账号体系
- **4 个 Tab 两年不变** — 与 Tony（腾讯创始人之一）的君子协议

### 4. 气质与价值观

- **产品要有灵魂**，像人一样有多个有机联系的"面"（骨骼、肌肉、气质、反应、谈吐）
- **"价值观决定产品特性"** — 不是功能分析决定，是价值观选择：
  - 不做"已读" — 保护隐私、减轻社交压力
  - 不做"iPhone 在线" — 不制造等级感
  - 不做 PC 版（早期） — 保护"永远在线"的信赖感
  - 不做滤镜（反思中） — 真正的生活状态不应该有滤镜
- **"是作品，而不是产品"** — 作品渴望完美，可打败功利产品
- **"工匠，而不是设计师"** — 瞄准 KPI 就不对了

### 5. 面向未来

- **"世界是新的"** — 忘记过去的数据和经验
- **二维码是手机时代的入口** — 连接现实与手机的桥梁
- 对未来场景的理解比当前需求更重要
- 每个时代有每个时代的产品（QQ → 微信）

## 设计方法论

### 需求来源优先级

1. **对群体效应的好奇**（最高）— 做之前不知道会发生什么
2. **自身的需要和周边朋友的反馈** — 以己推人
3. **对当下生活潮流的理解** — 从微博感知真实用户
4. **对信息流的理解** — 微信应成为 Push 中心
5. **对未来的理解** — 比当前需要更重要

### 核心原则

| 原则 | 内容 |
|------|------|
| **"爽"胜过功能** | 口碑传播的关键是让用户说"好玩"，而非"省钱" |
| **面向场景做设计** | 功能脱离场景没有意义；语音自动播放场景分析 |
| **保留变化** | 不在 1.0 做 2.0 的事；朋友圈第一版无评论回复 |
| **没有自然增长就不推广** | KPI 只是好产品的副产品 |
| **保持粗放笨拙** | 没有好方案时就先不做 |
| **操作的响应速度是第一体验** | "简单"+"快捷"="简捷" |
| **让功能存在于无形之中** | 商家会员扫二维码才出现 |
| **避免战略行为替代真实需求** | 导入 Qzone 照片破坏朋友圈氛围 |

### UI 原则

- **条理清晰最重要** — 比绚丽重要。用系统控件，保持一致
- **每个界面只有一个主题** — 两分式设计是不好的
- **需要文字解释的功能不是好体验** — 最好不用 Tips
- **隐藏数字** — 不精确显示百分比或联系人数量
- **禁止使用竞品** — 避免 UI 设计师受其影响

---

## 关于创新

- 所有创新都来自 **知识积累**，不存在凭空创造
- **"如果你说我是错的，最好证明你是对的"** — 回应抄袭指责
- 跟随用户而非跟随业界产品。用产品说话比辩解更有威力

---

## 与 KPI 文化的关系

张小龙主张 **跳出 KPI 做产品**，认为 KPI 只是好产品的副产品。如果工作围绕 KPI 指标，会变得无趣且失去发现。"好的产品应该是自然增长的，而不应该为 KPI 而改变产品。"如果产品经理的工作只围绕 KPI，"会非常无趣，会觉得每天工作得没有乐趣、没有发现。也不可能通过工作了解自己，了解别人，了解人性。"

---

## 结尾警示

> **"我所说的，都是错的。"**

张小龙强调不要将方法论当教条——"任何东西一旦说出来，变成了一个教条，它就已经错了。一定是你自己理解到的才是你自己的。"
