# Kimi K2 模型

## 这是什么

Kimi K2 是由月之暗面（Moonshot AI）开发的开放权重大语言模型系列，包括基础版 **Kimi K2** 和推理增强版 **Kimi K2 Thinking**。

## 核心架构特点

- **1T 参数 MoE**：61个层、384个专家、每 token 激活8个，160K 词汇量
- **原生 INT4 量化**：训练过程中即考虑量化，实现推理延迟和 GPU 内存的无损耗减少
- **256K 上下文窗口**：支持长文本和代码库处理
- **交错思维链**：在显式推理和工具使用之间交替，实现 200-300 步的稳定工具调用

## 能力特点

- **优势**：长周期 agent 任务的持久性，一次性编码任务表现出色
- **弱点**：无工具辅助时核心智能落后于 GPT-5，现实世界性能"时好时坏"（hit-or-miss）
- **安全问题**：原始安全得分 1.55%，存在越狱、操纵、骚扰等严重风险

## 战略意义

Kimi K2 证明了开放权重模型可以在 agent 持久性这一关键维度上与闭源巨头竞争，为全球 AI 竞赛设定了新基线。但它也暴露了闭源公司在安全对齐上投入的价值和成本。
