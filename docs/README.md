# Piki 文档目录

这个目录用于集中管理 Piki 的产品文档与开发过程文档，避免资料散落在仓库根目录。

## 目录结构

- `product/`
  - 产品定位、MVP 范围、路线图、补充产品笔记
- `development/`
  - 开发过程总览、阶段性结论、实现过程中的问题与解法
